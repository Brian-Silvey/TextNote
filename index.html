Compose music with text!

Instructions:

1. Type "BPM=" followed by the beats per minute desired.
2. Type "Type=" followed by the type of notes.

Types = 1, 1/2, 1/4, 1/8, 1/16

3. Type the name of the instrument you want to use followed by ": ".
4. Type notes in the format [note][modifier][number].
[note] = c, d, e, f, g, a, or b
[modifier] = -, ^, or _

- = standard note
^ = sharp note
_ = flat note

[number] = 0, 1, 2, 3, 4, 5, 6, 7, or 8

Note syntax example 1: c-4 = 4th C, or middle C.
Note syntax example 2: f_3 = 3rd F flat.

Every 3 spaces = 1 specified note. Note: the lowest possible note is c-0, and the highest possible note is b-8.

Instrument options: Piano
